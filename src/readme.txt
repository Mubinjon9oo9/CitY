This project consist from two programs: Server and Client. 
Sever getting and setting data to DB, and getting commands from client by TCP connection.

///////////

Steps to start using this project:
  1.  Start Server/Main.java
  2.  Start Client/Main.java
  3.  Type HELP in command line of Client/Main.java and get info about commands
